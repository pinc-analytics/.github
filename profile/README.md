## 📂 Featured Open Source & Projects

* 📡 **[AIS-catcher](https://github.com/pinc-analytics/AIS-catcher)** - Advanced AIS receiver for RTL-SDR dongles, Airspy, HackRF, SDRplay, and SoapySDR.
* 🦀 **[nmea-parser](https://github.com/pinc-analytics/nmea-parser)** - High-performance Rust NMEA 0183 sentence parser for AIS and GNSS messages.
* 🗺️ **[aismapping](https://github.com/pinc-analytics/aismapping)** - Python utility for plotting AIVDM NMEA data directly onto Google Maps.
* ☁️ **[service-screener-v2](https://github.com/pinc-analytics/service-screener-v2)** - Tool for evaluating AWS service configurations based on cloud architecture best practices.
